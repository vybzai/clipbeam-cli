clipbeam

Beam files, screenshots, and clipboard straight into your remote agent's reach.

Over SSH or Tailscale. No cloud, no account. Built for coding agents — every command is --json-able and self-describing.

Install

brew install vybzai/tap/clipbeam
# or
curl -fsSL https://raw.githubusercontent.com/vybzai/clipbeam-cli/main/install.sh | sh

Use

# once per box — bootstraps the remote binary over your existing SSH
clipbeam setup root@my-box

# then, from your laptop
clipbeam send screenshot.png my-box   # → prints the remote path
clipbeam shot my-box                  # screenshot → box

# on the box, your agent just reads the path
claude "what's in $(clipbeam last)"

Why

You can't paste an image into a TUI over SSH — the terminal reads your local clipboard, and nothing crosses the SSH boundary. The only thing a remote agent can ingest is a real file on the box's disk it can read by path. clipbeam puts it there and prints the path.
